# Wiki Schema

Generated by LM Wiki setup on 2026-04-30.
Update both claude.md AND agents.md whenever this schema changes.

## Folder Structure

| Folder | Role |
|--------|------|
| `wiki/` | All LLM-maintained wiki pages (vault root, alongside PARA) |
| `wiki/Synthesis/` | `/query` output — **READ ONLY for `/ingest`** (trust boundary) |

## Page Types

| Type | Purpose | Naming | Frontmatter? |
|------|---------|--------|--------------|
| `concept` | A methodology, framework, or idea | Title Case.md | Required |
| `entity` | A person, tool, organization, or project | Title Case.md | Required |
| `index` | Auto-generated catalog — regenerated every ingest | `index.md` | None |

## Required Frontmatter (concept and entity pages only — NOT index.md)

```yaml
---
type: concept | entity
sources: []        # vault-relative paths of raw sources that shaped this page
updated: YYYY-MM-DD
---
```

## Optional Frontmatter

```yaml
open_questions:    # list of unresolved questions this page raises
  - "..."
```

## Naming Conventions

- File names: Title Case, spaces allowed (e.g. `Attention Mechanism.md`)
- Qualifier for name collisions: `Transformer (Architecture).md`
- Backlinks: `[[Page Name]]` — must match file name exactly (no extension)
- `index.md`: regenerated on each ingest; NOT subject to idempotency check
- `Synthesis/*.md`: written by `/query` only; never modified by `/ingest`

## Special Files

| File | Written by | Rule |
|------|-----------|------|
| `index.md` | `/ingest` | Regenerated unconditionally on every ingest |
| `log.md` | `/ingest`, `/query`, `/lint` | Append-only timeline |
| `Lint-Report-YYYY-MM-DD.md` | `/lint` | Dated report file, not a permanent page |

## Log Format

Each entry starts with `## ` for grep-parseability:

```
## [YYYY-MM-DD] ingest | <source title or filename> | N pages updated, M created
## [YYYY-MM-DD] query | <question summary> | filed: wiki/Synthesis/Title.md OR not-filed
## [YYYY-MM-DD] lint | N pages checked | N contradictions, N orphans
```

Grep examples:
```bash
grep "^## " wiki/log.md | tail -5     # last 5 entries
grep "\] ingest" wiki/log.md           # all ingests
grep "\] lint" wiki/log.md             # all lint runs
```
